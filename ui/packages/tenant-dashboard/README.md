# Tenant Dashboard

Tenant-scoped self-service dashboard for Hybrid Sovereign Cloud.

## Features

- Entity-scoped resource lists filtered by namespace
- Self-service create forms for Teams, Projects, CloudOSO, CloudAWS, and Assignments
- RBAC-aware action visibility via `useCanI` hook

## Development

```bash
npm run dev -w @hybridsovereign/tenant-dashboard
```

## Build

```bash
npm run build -w @hybridsovereign/tenant-dashboard
```
