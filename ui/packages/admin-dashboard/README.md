# Admin Dashboard

Platform operator dashboard for Hybrid Sovereign Cloud.

## Pages

- Overview, Entities, Teams, Assignments, Projects, Personas
- Platforms, Clouds, Migrations, Operators, Services

## Development

```bash
npm run dev -w @hybridsovereign/admin-dashboard
```

## Build

```bash
npm run build -w @hybridsovereign/admin-dashboard
```
